js_adv_functions_lab
====================

Lab for javascript advanced functions

### Files
  * simple_iter.js - Simple DOM iteration.
  * simple_callback.js - Simple DOM iteration with callback.
  * anon_callback.js - Simple DOM iteration with anonymouscallback.
  * scoped_callback.js - Simple DOM iteration with callback that is an object's method. How 'this' is used in callbacks.
